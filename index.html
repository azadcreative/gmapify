What is GMapify

GMapify is a jQuery plugin for finding Schema markup (Organisation / Localbusiness / Place) and generating a neat Google Map widget.

What is Schema markup?

Schema markup is a semantic vocabulary for HTML that provides extra information for search engines. Schema tell search engines what your data means, which changes the way search listings are displayed with rich snippets and meta information extracted from your pages.

Organisation/Localbusiness/Place Schema markup tells the search engines information about your business, its location, branches, stores, etc.

What does GMapify do?

GMapify uses Schema markup to extract information from your webpages and generates a Google Maps widget with multiple markers, titles, descriptions.

The PostalAddress component has a number of parts, such as name, streetAddress, addressLocality, postalCode, addressRegion, addressCountry, description and openingHours. Use whichever is appropriate. Format is provided below:

    JavaScript

    The following are the defaults:

    {% highlight javascript %}$('body').gmapify({ canvas: "#map-canvas", meta: "#map-meta", metalist: 'li', zoom: 12, scrollwheel: false });{% endhighlight %}

    Options are as follows:

    Name Purpose and expected input
    canvas The ID where Google Maps would render
    meta Information source for the map
    metalist Is it ordered/unordered list item or description list or paragraph?
    zoom Default zoom level of the map
    scrollwheel Boolean. Whether map would allow the scrollwheel
